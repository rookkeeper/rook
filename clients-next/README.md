# clients-next

Parallel Apple-client rebuild area.

This tree exists so the next-generation iPhone, macOS, and shared RookKit code can grow from simpler first principles without clobbering the current clients.

Planned directories:
- `RookKit/` — shared Swift layer for anything that should behave the same on iPhone and macOS
- `mac/` — macOS-specific app shell and integrations
- `iphone/` — iPhone-specific app shell and integrations
